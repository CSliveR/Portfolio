Website for an IT service company

Project description

For this project, two of my classmates and I had to make the CGI IT service company website targeting third grade students who could be interested by an internship in this company. This project was divided into three parts: requirements gathering where we collected information about the company, the design where we built the website model and the implementation where we developed the website through coding.

Skills acquired

Summarizing information to tailor it to the target audience.

Making a model with a design tool.

Creating a website in collaboration.

Setting up tools to communicate within the team.
