Exploitation d'une BD en SQL

Description du projet

Pour ce projet, je devais réaliser une extraction de données sur une base de données open source (celle d'openFood Fact) qui recensait plusieurs centaines de produits avec diverses informations nutritionnelles ainsi que leurs nutriscores. Il a fallu explorer les données, supprimer les données mal renseignées, trouver les données intéressantes pour répondre au problème, manipuler ces données puis les extraire dans un fichier CSV. Ensuite, il a fallu réutiliser ces données pour faire des graphiques en utilisant RStudio et mener une analyse mathématique entièrement rédigée en anglais.

Compétences acquises

Exploration et analyse des données

Interrogation de la base de données

Nettoyage des données erronées ou incomplètes

Sélection de données pertinentes

Exportation des données sous forme de fichiers CSV exploitables

Création de graphiques avec RStudio pour visualiser et analyser les résultats

Analyse complète des résultats en anglais
