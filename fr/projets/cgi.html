Site web pour une ESN

Description du projet

Pour ce projet, deux de mes camarades et moi devions réaliser le site web de l'entreprise de services numériques CGI à destination d'élèves de troisième qui seraient intéressés par un stage dans cette ESN. Ce projet s'est découpé en 3 phases : le recueil des besoins où nous avons recueilli des informations sur l'entreprise, la conception où nous avons construit la maquette de notre site web et la réalisation où nous avons codé le site web.

Compétences acquises

Synthétiser des informations pour les adapter au public visé.

Réaliser une maquette avec un outil de conception.

Réaliser un site web en collaboration.

Mettre en place des outils pour communiquer au sein de l'équipe.
