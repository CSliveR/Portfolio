Jeu de bataille de cartes

Description du projet

Ce projet est un jeu de bataille de cartes guidé par un pdf trouvé sur internet et une vidéo explicative qui détaille le déroulement interne du jeu. Les ressources utilisées sont précisées dans le repository du projet sur github. Ce projet m'a permis de davantage développer mes compétences en programmation objet et de manipuler des objets de manière aléatoire.
