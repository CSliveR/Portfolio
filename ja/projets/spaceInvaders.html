Space invaders

Project description

This project is a simplified Space Invaders. There is a menu where you have to press the 'S' key to start playing. The goal of the game is to avoid the falling fireballs with the ship by using directionnal arrows to survive as long as possible against a random number of fireballs waves. If the ship is hit by a fireball, the game is lost. However, if the ship was never hit and all the waves were passed, the game is won. We can restart a game by pressing the 'C' key to play again. This project may have improvements in the future.
